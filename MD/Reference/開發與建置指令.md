# 開發與建置指令

## 建置（輸出到 Build/）
```bash
# 建置 App（會自動帶入 Core 與原生 SDL2.dll）
dotnet build cs_ported/App/App.csproj -c Release -o C:\ai_project\AprCSTyrian\Build
```
- 勿對 `.slnx` 整個 solution 用 `-o`，會觸發 `NETSDK1194` 警告（多專案輸出到同一目錄）。

## 執行
```bash
# 直接執行
C:\ai_project\AprCSTyrian\Build\AprCSTyrian.exe
# 指定資料根目錄（預設為執行檔旁的 data\）
C:\ai_project\AprCSTyrian\Build\AprCSTyrian.exe <資料夾路徑>
```
- WinExe 無 console；致命錯誤寫入執行檔旁 `crash.log`。

## 還原套件
```bash
dotnet restore cs_ported/AprCSTyrian.slnx
```
