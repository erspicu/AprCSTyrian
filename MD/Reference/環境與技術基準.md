# 環境與技術基準

- **目標框架**：.NET 10（`net10.0`）。已安裝 SDK：`10.0.109`（另有 8/9/11-preview）。
- **語言**：C#（啟用 `nullable`、`ImplicitUsings`、`AllowUnsafeBlocks`）。
- **平台**：Windows 11（主要開發）；程式碼盡量保持跨平台，平台相依只進 `App`。
- **SDL 繫結**：`Sayers.SDL2.Core` 1.0.11
  - SDL2-CS 的 NuGet 包裝，namespace `SDL2`、static class `SDL`，與原始 C 的 SDL2 呼叫 1:1。
  - 內含 win_x64 原生 `SDL2.dll`（及 image/mixer/ttf 與編解碼器），`.targets` 自動複製到輸出。
  - 因原生庫僅 win_x64，`App` 設 `PlatformTarget=x64`。
- **建置輸出**：`Build/`（單一執行檔 `AprCSTyrian.exe` + 原生 DLL）。
