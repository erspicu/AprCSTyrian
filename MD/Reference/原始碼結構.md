# 原始碼結構（sources/）

OpenTyrian upstream（clone 自 github.com/opentyrian/opentyrian），約 50 個 `.c` 模組 + 對應 `.h`。

## Pascal 型別對應
來源為 C，但型別命名沿用 Pascal 慣例（Tyrian 原為 Turbo Pascal）。移植對應：

| 原始 | 寬度 | C# |
|------|------|-----|
| `JE_longint`  | Int32  | `int` |
| `JE_integer`  | Int16  | `short` |
| `JE_shortint` | Int8   | `sbyte` |
| `JE_word`     | UInt16 | `ushort` |
| `JE_byte`     | UInt8  | `byte` |
| `JE_boolean`  | bool   | `bool` |
| `JE_real`     | float  | `float` |

## 相依函式庫
- **SDL2 + SDL2_net**（視窗、輸入、音訊、UDP 網路）
- OPL FM 音源模擬、VGA 256 色調色盤、PCX / 自訂格式圖檔

## 模組分類（節選）
- 進入點/主流程：`opentyr`、`mainint`、`tyrian2`
- 繪圖：`video`、`vga256d`、`sprite`、`font`/`fonthand`、`palette`、`backgrnd`、`starlib`
- 音訊：`loudness`(混音)、`opl`(FM)、`lds_play`、`nortsong`、`jukebox`、`musmast`、`sndmast`
- 資料載入：`file`、`lvllib`、`pcxload`、`picload`、`animlib`
- 設定/存檔：`config`、`config_file`、`memreader`/`memwriter`
- 遊戲邏輯：`player`、`shots`、`destruct`、`editship`、`episodes`
- 輸入：`keyboard`、`joystick`、`mouse`
- 工具：`mtrand`(亂數)、`params`/`arg_parse`
- 選單：`menus`、`game_menu`、`helptext`
- **網路：`network`（不移植）**

## 資料檔
遊戲需 Tyrian 2.1 freeware 資料檔（非程式碼）：https://camanis.net/tyrian/tyrian21.zip
放入 `Build/data/` 或以執行參數指定資料根目錄。
