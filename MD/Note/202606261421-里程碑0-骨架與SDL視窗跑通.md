# 里程碑 0：雙專案骨架 + SDL 視窗跑通

日期：2026-06-26 14:21

## 成果
建立 Ports & Adapters 雙專案骨架並驗證 Core ↔ SDL 管線打通。

- `cs_ported/AprCSTyrian.slnx`、`Core`（class lib，零 SDL）、`App`（WinExe，參考 Core）
- SDL 繫結：`Sayers.SDL2.Core` 1.0.11（SDL2-CS 包裝，內含原生 SDL2.dll，自動複製到輸出）
- Core Ports：`IVideoBackend/IAudioBackend/IInputBackend/IClock/IFileSystem/IGamePlatform`
- Core 型別：`Color`(8-bit RGB+FromVga)、`GameKey`、`VgaScreen`(320×200)
- SDL Adapters：`SdlVideo`（window+renderer+streaming texture，整數縮放）、`SdlInput`、
  `SdlClock`、`SdlAudio`（callback）、`PhysicalFileSystem`、`SdlPlatform`
- `Program.cs` 組合根；WinExe 無 console，致命錯誤寫 `crash.log`
- `TyrianGame.Run()` 顯示動態 HSV 測試圖樣，ESC/X 關閉

## 驗證
- `dotnet build App/App.csproj -c Release -o Build`：**0 警告 0 錯誤**
- 實機執行 4 秒（timeout 自動關閉）：exit 124（仍在執行被殺）、**無 crash.log** → 視窗開啟、迴圈正常

## 注意事項
- 建置用 App 專案 + `-o`，勿對 .slnx 用 `-o`（NETSDK1194 警告）。
- 原生庫僅 win_x64，App 已設 `PlatformTarget=x64`。
